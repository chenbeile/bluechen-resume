基于Vue & Vuex移动端旅游项目
预览地址 - 源码展示
功能介绍：首页城市与城市选择页面数据共享，多区域列表的循环展示，多区域轮播功能，城市选择页面有城市的展示，城市的搜索功能以及城市右侧字母与左侧区块动画联动的效果，景点详情页面有公共画廊组件，渐隐渐现的header组件递归展示的列表组件
技术细节：项目的矢量图标使用的是阿里提供的iconfont，可以避免调整高度和位置的麻烦，使用webpack配置简化代码，使用stylus预处理css，轮播图使用的是vue-awesome-swiper插件实现首页图片区域、下方图标区域的轮播以及页面详情公共画廊图片的轮播，数据请求使用的是axios，可以避免多次请求数据。每次跳转页面，首页点击header中的城市跳转到城市列表时，每次都会重新请求数据，为此使用了keep-alive取缓存的特性来优化性能，使用vue-devtools工具来进行vue项目的调试
使用技术：vue@2.4 + webpack + node + vue-router（多页面间路由） + vue-cli + stylus（编写前端样式） + iconfont + fastclick + vue-awesome-swiper（页面上轮播效果） + axios（ajax数据的获取） + better-scroll + vuex（各组件间数据共享）
